# HKDSE 平面鏡反射互動實驗室

## 在 Google Chrome 開啟
雙擊 `Open Plane Mirror Lab.bat`，或在 Chrome 開啟 `index.html`

## 場景（5 個分頁）
| 分頁 | 題目 | 預設結果 |
| 像的形成 | 基本虛像 | u = v |
| 最小鏡長 | Q21 | 0.8 m |
| 最小鏡高 | Q22 / Q17 | 0.75 m / 0.50 m |
| 看後方物件 | Q23 / Q24 | 1.83 m / 2.6 m |
| 夾角鏡 | 45 deg | 7 個像 |

## 動畫
- 初始不顯示光線
- **逐步繪製光線**：自動播放
- **下一步**：單步
- **顯示全部**：一次顯示

## 修改程式後
在 repo 根目錄執行 `npm run build:plane-mirror-lab`，或於本目錄執行 `node build-bundle.cjs`。
