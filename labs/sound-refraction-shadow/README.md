# Sound Refraction Lab (Shadow Zone)

Editable source for the sound refraction / atmospheric shadow-zone interactive lab.

## Where to edit (on GitHub)

Edit files in this folder on GitHub (or clone the repo). Changes here are the source of truth — do **not** edit only a local Downloads copy.

- `index.html` — full lab UI + physics (night inversion vs hot day, ray bending, TIR / shadow zone)

## How it is used

- Standalone: open `index.html` in a browser (via the Vite/dev server or GitHub Pages).
- Embedded in the Waves Hub: iframe with `?embed=1&lang=en|zh`.

After editing, rebuild (`npm run build`) and push so GitHub Pages picks up the update.
