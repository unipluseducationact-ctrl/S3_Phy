# Double Slit & Diffraction Grating Compare Lab

Editable source for the double-slit vs diffraction grating comparison lab（雙狹縫與光柵）.

## Where to edit (on GitHub)

Edit files in this folder on GitHub (or clone the repo). This is the source of truth — do **not** edit only a local Downloads copy.

- `index.html` — full lab UI (tabs, diagrams, MathJax)

## How it is used

- Standalone: open `index.html` via the Vite/dev server or GitHub Pages.
- Embedded in the Waves Hub: iframe with `?embed=1&lang=en|zh`.

After editing, rebuild (`npm run build`) and push so GitHub Pages picks up the update.
