# Ultrasound Reflection Lab

Editable source for the Slow Motion Ultrasound Reflection interactive lab.

## Where to edit (on GitHub)

Edit files in this folder on GitHub (or clone the repo). Changes here are the source of truth — do **not** edit only your local Downloads copy.

- `index.html` — full lab UI + animation (source → moving point object → echo)

## How it is used

- Standalone: open `index.html` in a browser (via the Vite/dev server or GitHub Pages).
- Embedded in the Waves Hub: iframe with `?embed=1&lang=en|zh`.

After editing, rebuild (`npm run build`) and push so GitHub Pages picks up the update.
